Jandi 서비스는 블로그의 글을 잔디밭으로 보여주고 주제 분석을 하여 블로그를 분석하는 서비스입니다.
### 기능
- 블로그 등록/삭제
- 사용자 글 주제 분석 
- 잔디밭 조회
- 잔디밭 iframe export
### 기술스택
- Backend: **FastAPI**
- Frontend: React (with Figma Make)
- Db: **Postgresql** (Cloud SQL)
- Message Queue: **RabbitMQ**
- Deployment: Docker, K8s, Helm, Prometheus, Grafana, Terraform, ...
